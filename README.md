Welcome to FibonacciNumberCalculator, NodeJS CRUD Application to get Fibonacci Number

Steps to Run Application:

Run command - npm i
Make .env file
Run npm start

POSTMAN API's:

getFibonacciNumber - http://localhost:3000/fibonacci/:index

TEMPLATE FOR DOTENV FILE:

PORT
REDIS_HOST
REDIS_PORT
DB_USER
DB_PASS
DB_URL
